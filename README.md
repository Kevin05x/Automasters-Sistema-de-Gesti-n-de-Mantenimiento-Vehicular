# 🚗 Automasters: Sistema de Gestión de Mantenimiento Vehicular

> Una solución robusta de escritorio desarrollada en **WPF** y **.NET 8** para la administración centralizada de servicios mecánicos, repuestos y control de personal.

---

## 📖 Resumen del Proyecto
Automasters está diseñado para optimizar la operatividad de talleres y centros de servicio automotriz. El sistema permite el seguimiento detallado de mantenimientos, gestión de inventario de repuestos y administración de proveedores, todo bajo una arquitectura escalable y una interfaz de usuario intuitiva.

## 🛠️ Stack Tecnológico

| Capa | Tecnología |
| :--- | :--- |
| **Interfaz de Usuario** | WPF (Windows Presentation Foundation) / XAML |
| **Lógica de Negocio** | .NET 8 SDK |
| **Acceso a Datos** | Entity Framework & SQL Client |
| **Base de Datos** | SQL Server |
| **Herramientas** | Visual Studio 2022 |

## 🏗️ Arquitectura del Sistema
El proyecto implementa una separación de responsabilidades mediante una estructura de tres capas:
* **`MantenimientoUI`**: Capa de presentación con recursos visuales y ventanas.
* **`BusinessLogic`**: Reglas de negocio y validaciones de servicios.
* **`Data`**: Modelado de entidades y persistencia en base de datos.
